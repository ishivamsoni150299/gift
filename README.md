# Digital Engagement Gift

This is a static digital gift page for the engagement on 17 July 2026.

## Personalize it

1. Replace `assets/fiancee-photo.svg` with her real image, or send me the image and I will wire it in.
2. Edit the names, message, and memories in `index.html`.
3. Open `index.html` in a browser to preview.

## Make the scannable code

Publish this folder to any static host, then open `qr.html`, paste the final live URL, and print the QR code for the gift box. A QR code is better than a traditional barcode for a web link because phone cameras can open it directly.
